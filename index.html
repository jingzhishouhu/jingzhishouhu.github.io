MySQL Redis MongoDB
类型： 关系型 非关系型 非关系型
端口： 3306 6379 27017
进入方式： mysql -uroot -p redis-cli mongo
查看所有库： show databases; 16个库，默认在0 show dbs
查看当前库： select database(); 能看到 db/db.getName()
选择库： use 库名; select 数字 use 库名 存在切换，不存在创建
创建库： create database 库名; 不用创建 use 库名
删除库： drop database 库名; flushdb/all 清空 db.dropDatabase() 删除当前库
